WDD 130

Personal Site Child 1 page

Last week you should have gotten all most of the styling for your Home page done. This week we will style the first Child page.

Activity Directions

  1. Create the file

    Create a new HTML file for your child page. Name it something short and that relates to what the page is. Remember to avoid spaces and capitols in your name. You may want to then Copy/paste the contents of your Home page into this new file, replacing what was there. Then remove the Home page specific content (usually the contents of main).

  2. Child Page Grid

    Open the wireframe for your first child page. We need to build the Grid for your layout. From the wireframe determine how many columns and rows you will need. Then create those with CSS. Then start placing your elements into the grid cells.

  3. Spacing and Alignment

    Add padding and margins where appropriate to give your layout the right amount of white-space.

    Adjust the alignment of elements as well at this time as needed.

  4. Link it up

    Make sure to update the navigation in both your Home page and the child page so that if you click on a link in either page it will take you to the other.

  5. Publish

    Publish your site and submit the link to Ilearn. Let the instructor know which child page you completed.

Grading

  1. Child page layout looks like the Child page wireframe in the site-plan. (5pts)
  2. Colors from the site-plan used. (5pts)
  3. Font(s) from the site-plan used (5pts)
  4. The Child page is visually appealing. Attention has been given to Whitespace, Alignment, Contrast in particular. (5pts)
