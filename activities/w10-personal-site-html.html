    Activity Directions

    1. Home Page wireframe

      Open the wireframe for your homepage first. You may even want to print it out and grab a marker and a ruler. We need to visualize any groups of content on the page and draw boxes around the groups. Each of these boxes should be represented as HTML in our page. div, section, main, header, footer, nav, etc. elements should be used for this purpose.

      Create an index.html file in your repository, (folder containing your personal website), for your home page.

    2. Home Page HTML and Content

      Once you have identified any content groups on your wireframe, begin at the top left corner and start adding the elements necessary, then add the content for the home page.

      Add a link to your siteplan to the footer of the Home page.

    3. Continue for page 2 and 3

      Once you have the home page done, do the same exercise for pages two and three. The header and footer code should be very similar if not identical to the home page. Most of your changes should happen in the <main> element.

      You should create a new HTML file for each page. Name them with something meaningful (but without any spaces or capitalization). You may even want to copy/paste the HTML from your index.html file as a starting point for these child pages. Just remember that while you will want to keep the code in the <header> and <footer> (since it should be the same across all three pages), you should clear out the <main> element when starting each new child page.

    4. Publish

      Publish your site and submit the link to Ilearn.
