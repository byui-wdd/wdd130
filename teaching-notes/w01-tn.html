Week 01

Before the Lesson Begins

  • Students will have a preparation quiz in the Intro Week. Usually this would be done before the first day of class, but this week they will complete it before the second day of class.
  • Preparation Quiz results.  The students must complete the quiz before the first day of class for weeks 2-7. The due date and time should be set for before the first class of the week. This is to encourage them to spend the time needed to really learn those terms. They can take it as many times as they need.
  • Canvas might be a new experience for many students. You may want to lead them through a quick tour.
  • Remind students to plan their time so that they can work consistently on the course throughout the week.
  • Many students are fresh out of high school; make the expectations and standards clear to the students.

During the Lesson

  • Install Visual Studio Code and some extensions in class with the students. Here is an example of what you'd do in class (Steps 2-4). Software Setup to do in Class

  • Make sure students actually set up the aboutme folder with an index.html started, and an images folder. This is demonstrated on the video.

  • Take some time during class to build a simple page together so they feel comfortable with the 'Getting to Know You' assignment.  For example I might build something like this HTML Practice Example

Assessments

  • Students will create their first html page all about them and place it on a discussion board this week for you and the other students to see.

[Note] 

  • Remind students that they will have a reading assignment and quiz to take before the first class of each week. The videos and/or reading and the related quiz, are in the Prepare section of each week.
